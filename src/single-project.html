Название проекта

Сжатие видео потока кодеком MJPEG в максимальном разрешении без ограничения по битрейту позволяет получить полноразмерные стоп-кадры без потери качества для распознавания и идентификации объектов. MJPEG предъявляет минимальные требования к вычислительным ресурсам системы, что уменьшает нагрузку на сервер и серверное ПО. При использовании видеоаналитики, например “Трекер объекта”, поиск объекта по цвету, очень важной составляющей является цветопередача видеоизображения. В IP-камерах Rubezh настройка среднего уровня яркости позволяет получить более естественную цветопередачу по сравнению с IP-камерами, где такой функции нет.
